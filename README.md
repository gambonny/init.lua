# nvim-conf

### Shortcuts
    jj         => esc
    \d         => delete buffer
    <leader>ev => edit init.lua
    <leader>w  => save
    <leader>x  => save and quit
    <leader>q  => quit
    <leader>Q  => quit all
  
### System Copy

    cpiw => copy word into system clipboard
    cpi' => copy inside single quotes to system clipboard
    cvi' => paste inside single quotes from system clipboard
